# Medical Recommendation System

AI-powered medical health assessment and recommendation system.

## Quick Deploy to Render (Easiest - Free)

1. Push your code to GitHub
2. Go to [render.com](https://render.com) and sign up (free)
3. Click "New" â†’ "Web Service"
4. Connect your GitHub repo: `Suyash-666/medical-recommendation-system`
5. Render auto-detects settings from `render.yaml`
6. Click "Create Web Service"
7. Done! Your app will be live in ~2 minutes

## Local Development

```bash
pip install -r requirements.txt
python app.py
```

Visit: http://127.0.0.1:5000

## Features

- User authentication (signup/login)
- Health prediction using ensemble AI models
- Medical recommendations (diet, precautions, medications)
- Prediction history tracking
- Demo mode (works without Firebase)
